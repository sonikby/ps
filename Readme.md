Папка Clients - абстракция
Внешнии папки - часть приложение

Жизненный цикл:
- В БД ищется транзакция 
- Полученными данными заполняется DTO InnerTransaction
- Создается соот-й платежной системе процессор (см. AbstractPaymentProcessor)
- Создается соот-я команда (напр. CreateTransaction)
- У команды выполняется метод execute
- Обработка ответа
